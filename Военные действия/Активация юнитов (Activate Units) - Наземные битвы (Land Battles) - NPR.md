## Цикл против NPR

1. Не забудьте про морские битвы
2. Рассчет сил NPR
	1. Выставляем число пехоты по MC (Military capacity) = NPR Tax value + Vassal tax value в области + граничащие с учетом портов в невраждебных зонах
	2. Если 3+, то тянем карточку из военной колоды для возможного бонуса по кубам
3. Подготовка (Battle Preparations)
	1. Атакующий может назначить генерала (за военный куб)
	2. **ИЛИ** **БЕСПЛАТНО** переназначить генерала из другой армии
	3. **ИЛИ** назначить правителя в качестве генерала (Ruler general)
	4. Защищающийся может назначить генерала (за военный куб)
	5. **ИЛИ** **БЕСПЛАТНО** переназначить генерала из другой армии
	6. **ИЛИ** назначить правителя в качестве генерала (Ruler general)
4. Карточки боевых действий (Battle Actions)
	1. Атакующий разыгрывает карточки
	2. Защищающийся NPR / DNPR не имеет таких карточек, но
	3. **ОПЦИОНАЛЬНО** все игроки могут помогать NPR / DNPR разыгрывая карточки боевых действий с руки
5. Боевые кубы (Roll Battle Dice)
6. Если число юнитов NPR < число юнитов атакующего, NPR отступает (исчезает), если:
	1. Область не содержит столицы NPR
	2. Область не содержит повстанцев

### Рассчет MC для NPR
1. Tax value NPR и вассалов в области
2. Tax value NPR и вассалов в соседних областях, которые не содержат враждебных юнитов
3. Tax value NPR и вассалов в областях, соседних по морю, если морская зона не враждебна

### Как уменьшить MC для NPR
1. Для морских NPR можно вывести корабль в морские зоны (все) и заблокировать ее для рассчета MC. Работает для морских NPR
  2. Для наземных можно воспользоваться тем, что области с враждебными юнитами не учитываются
